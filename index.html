Web Dev

Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.

SQL

SQL stands for Structured Query Language. SQL lets you access and manipulate databases. SQL became a standard of the American National Standards Institute (ANSI) in 1986, and of the International Organization for Standardization (ISO) in 1987.

Python

A versatile programming language widely used for its robust libraries (e.g., Pandas, NumPy) that enable efficient data manipulation, cleaning, statistical analysis, and machine learning. Its flexibility makes it ideal for handling diverse data tasks and building analytical workflows.

R

A powerful statistical programming language and environment specifically designed for statistical computing and graphics. R excels in advanced statistical modeling, complex data visualization (e.g., with ggplot2), and provides a vast ecosystem of packages for various analytical techniques.

Power Bi

A leading business intelligence tool that transforms raw data into interactive and insightful dashboards and reports. It enables data modeling, advanced data analysis using DAX, and the creation of compelling visual stories to support data-driven decision-making.

Tableau

A powerful and intuitive data visualization tool that allows for rapid exploration and discovery of insights from various data sources. It's renowned for its ability to create highly interactive dashboards and compelling visual analytics that simplify complex data for diverse audiences.
